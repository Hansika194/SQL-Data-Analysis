# SQL-Data-Analysis
SQL Data Analysis project demonstrating SQL queries, aggregations, joins, subqueries, views, and business insights

Overview

This project demonstrates SQL-based data analysis using a Student Management Database. The database is extended with Courses and Enrollments tables to analyze student performance, course participation, and academic results through various SQL queries.

Database Structure

Students

Stores student information:

* Student ID
* Name
* Age

Courses

Stores course details:

* Course ID
* Course Name

Enrollments

Stores enrollment records:

* Student ID
* Course ID
* Grade

SQL Queries Performed

1. Student Course Enrollment Report

Displays students, their enrolled courses, and corresponding grades.

2. Average Grade by Course

Calculates the average grade for each course using aggregate functions.

3. Top 3 Performing Students

Identifies the highest-performing students based on average grades.

4. Failed Students Analysis

Counts students whose grades are below the passing threshold.

Technologies Used

* SQL

Learning Outcomes

* Designed and managed relational database tables.
* Performed data analysis using SQL queries.
* Applied JOIN operations to combine data from multiple tables.
* Used aggregate functions such as AVG() and COUNT().
